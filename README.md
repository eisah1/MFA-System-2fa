# MFA Authentication System

A university software engineering project focused on implementing a secure multi factor authentication system using Java and Maven.

## Features

* User login and registration
* Multi factor authentication process
* Secure verification functionality
* Database integration
* Input validation and error handling

## Technologies Used

* Java
* Maven
* SQL
* Git

## Project Overview

This project was developed as part of university coursework to explore authentication systems, backend development and secure user verification processes. The application was designed to improve account security through multi step authentication methods and structured software development practices.
